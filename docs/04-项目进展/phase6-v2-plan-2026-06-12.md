## 🔧 技术约束
- **纯前端改动**：admin.html 单文件，Vue 3 CDN
- **后端最小改动**：api_server.py 仅加分页和批量 API
- **不引入外部图表库**：饼图用 CSS conic-gradient
- **不提交 data/*.json**

## 📊 当前 API 状态
- `GET /api/v1/admin/tenants` — 已有 q/plan_filter/status_filter，缺 page/page_size
- `POST /api/v1/admin/tenants` — 创建租户 ✅
- `PUT /api/v1/admin/tenants/{id}` — 编辑租户 ✅
- `DELETE /api/v1/admin/tenants/{id}` — 删除租户 ✅
- `GET /api/v1/admin/users` — 用户列表 ✅
- `POST/PUT/DELETE /api/v1/admin/users/{id}` — 用户 CRUD ✅
- 密码重置：后端有 `/api/v1/auth/reset-password`，前端缺入口
