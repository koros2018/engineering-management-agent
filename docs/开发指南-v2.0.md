  -keyout /etc/nginx/ssl/ema.key \
  -out /etc/nginx/ssl/ema.crt \
  -subj "/CN=localhost"

## 三、核心概念

### 3.1 Agent 架构

```
用户输入
  ↓
IntentClassifier → 识别意图 (classify intent)
  ↓
TaskPlanner → 制定执行计划 (plan steps)
  ↓
Orchestrator → 调度 Sub-Agent (dispatch)
  ↓
ResultCompiler → 整合结果 (compile)
  ↓
返回用户
```

### 3.2 添加新的 Sub-Agent

```python
# src/sub_agents/my_new_agent.py
from agent.base_agent import BaseAgent, AgentResult, Task

class MyNewAgent(BaseAgent):
    AGENT_ID = "my_new_agent"
    NAME = "我的新Agent"
    ICON = "🆕"
    DESCRIPTION = "新增的业务Agent"

    def get_supported_tasks(self) -> List[str]:
        return ["my_task", "chat"]

    async def execute(self, task: Task) -> AgentResult:
        if task.task_type == "my_task":
            return AgentResult(
                task_id=task.task_id,
                agent_id=self.AGENT_ID,
                status="success",
                output={"result": "done"},
                confidence=0.95,
            )
        return await super().execute(task)
```

然后在 `sub_agents/__init__.py` 注册：

```python
from sub_agents.my_new_agent import MyNewAgent

# 在 AgentManager 中添加
self.register(MyNewAgent())
```

最后在 `agent/intent_classifier.py` 添加意图关键词。

### 3.3 添加 API 端点

```python
# src/api_server.py
from pydantic import BaseModel

class MyRequest(BaseModel):
    param1: str
    param2: int = 1

@app.post("/api/v1/my-endpoint")
async def my_endpoint(
    req: MyRequest,
    user: dict = Depends(get_current_user),  # 强制认证
):
    result = do_something(req.param1, req.param2)
    return {"success": True, "data": result}
```

### 3.4 数据流

```
请求 → (Auth中间件) → (日志) → 路由处理 → 返回
                    ↓
              ChromaDB 存储对话历史
              通知引擎检查 (异步)
              邮件发送 (异步)
```

## 五、日志规范

```python
from tools.logging_utils import get_logger, set_request_context, log_api_request

logger = get_logger(__name__)

# 基础日志
logger.info("操作完成", extra={"extra_data": {"user_count": 42}})
logger.error("操作失败", exc_info=True)

# API请求日志（fastapi中间件中使用）
log_api_request("GET", "/api/v1/agents", 200, 5.2)

from tools.logging_utils import log_security_event
log_security_event("login_failed", "密码错误", severity="medium", user_id="user_123")
```

日志输出：
- `logs/ema-YYYY-MM-DD.log` — 结构化 JSON（INFO+级别）
- `stderr` — 彩色控制台（所有级别）
