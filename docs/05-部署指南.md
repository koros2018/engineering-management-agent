# EMA 部署指南

> 工程管理智能体 (Engineering Management Agent) v3.5.0
> Slogan: 工程管理，从"人管"到"智能体协管"

---

## 快速开始（3步）

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动服务

```bash
./manage.sh start
```

### 3. 打开浏览器

```bash
./manage.sh open
# 或直接访问: http://127.0.0.1:6189
```

## 服务管理

| 命令 | 说明 |
|------|------|
| `./manage.sh start` | 启动所有服务 |
| `./manage.sh stop` | 停止所有服务 |
| `./manage.sh restart` | 重启所有服务 |
| `./manage.sh status` | 查看运行状态 |
| `./manage.sh logs` | 查看实时日志 |
| `./manage.sh open` | 打开浏览器 |
| `./manage.sh test` | 运行全量测试 |

## 环境要求

- **Python**: 3.10+
- **内存**: 最低 512MB，推荐 2GB+
- **磁盘**: 最低 500MB
- **Ollama** (可选): 本地AI推理，不装也能用云端模型

### 可选：Ollama 本地模型

```bash
# 安装 Ollama
curl -fsSL https://ollama.com/install.sh | sh

# 下载模型
ollama pull deepseek-r1:8b
```

## 影子监督员

独立守护进程，每3分钟自动巡检：

```bash
# 查看状态
sudo systemctl status ema-supervisor

# 查看日志
sudo journalctl -u ema-supervisor -f

# 重启
sudo systemctl restart ema-supervisor
```

**巡检内容：**
1. 服务健康检查（API/UI/Ollama/NVIDIA）
2. LLM 透测（对话响应超时检测）
3. 会话锁死检测（>15min无响应自动告警）
4. 项目进度监督（未提交文件提醒）

**自动修复：**
- API 挂了 → 自动重启
- UI 挂了 → 自动重启

## Docker 部署（生产环境）

```bash
# 构建镜像
docker build -t ema:latest .

# 开发模式（API + UI）
docker-compose --profile dev up -d

# 生产模式（API + Nginx + UI）
docker-compose --profile prod up -d

# 查看状态
docker-compose ps

# 查看日志
docker-compose logs -f api
```

## 云平台部署方案

### 方案一：单云服务器（推荐试用阶段）

```
云服务器 (2核4G)
├── Docker Compose
│   ├── ema-api (6188)
│   ├── nginx (80/443)
│   └── UI 静态文件
└── 数据卷持久化
```

### 方案二：多服务分离（生产阶段）

```
云服务器A: API服务 (FastAPI + Uvicorn)
云服务器B: 前端UI (Nginx 静态文件)
云服务器C: AI推理 (Ollama / 云端API)
对象存储:   图纸文件 + 生成文档
```

## 常见问题

**Q: 端口被占用？**
```bash
# 查看占用
lsof -i :6188
lsof -i :6189

# 修改端口
./manage.sh stop
# 编辑 manage.sh 中的 API_PORT / UI_PORT
./manage.sh start
```

**Q: 测试失败？**
```bash
# 确认API在运行
curl http://127.0.0.1:6188/health

# 运行单测（不需要API）
python3 -m pytest tests/ -k "not E2E" -q

# 运行E2E测试（需要API）
python3 -m pytest tests/test_phase22_e2e.py -v
```

**Q: Ollama 模型下载慢？**
```bash
# 使用国内镜像
export OLLAMA_HOST=http://your-mirror:11434
ollama pull deepseek-r1:8b
```
