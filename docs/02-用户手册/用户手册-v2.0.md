# EMA 工程管理智能体 — 用户手册 v2.0

> **访问地址**: `https://localhost/ui/index.html`
> **口号**: 工程管理，从"人管"到"智能体协管"

## 一、快速开始

### 1.1 系统要求

| 环境 | 最低要求 |
|------|---------|
| 浏览器 | Chrome/Firefox/Edge/Safari 最新版 |
| 移动端 | iOS 14+ / Android 10+ (支持PWA安装) |
| 微信小程序 | 微信 8.0+ |

### 1.2 首次登录

1. 打开浏览器访问 `https://localhost/ui/index.html`
2. 自动跳转登录页，选择登录方式：
   - **密码登录**: 输入用户名和密码
   - **微信扫码**: 点击微信图标，扫描二维码登录
3. 新用户点击「注册账号」，填写信息后自动登录

### 1.3 添加到桌面 (PWA)

- **iOS Safari**: 点击分享按钮 → 「添加到主屏幕」
- **Android Chrome**: 菜单 → 「添加到主屏幕」
- 安装后可离线查看历史对话和通知

## 二、工作台使用

### 2.1 智能体选择

左侧边栏展示可用的智能体：

| Agent | 图标 | 功能 |
|-------|------|------|
| 技术研发中心 | 🔧 | 图纸解析、分类、工程量提取、设计优化 |
| 安全与合规中心 | 🔒 | 国标审图（消防/结构/电气）、合规审查 |
| 市场与销售中心 | 📡 | 投标文件生成、智能报价、市场分析 |
| 工程交付中心 | 🏗️ | SOP/MOP/EOP/LCC 全生命周期文档 |
| 成本效益中心 | 💰 | 工程量清单、造价预算、成本分析 |
| 客户服务中心 | 🤝 | FAQ问答、培训材料、用户反馈 |

**操作**: 点击Agent切换 → 在底部输入框发送消息。

### 2.2 图纸上传分析

1. 切换到「技术研发中心」
2. 点击输入框上方的 📎 附件按钮
3. 选择 DWG/DXF/PDF 文件
4. 系统自动完成：
   - 图层识别 → 图纸类型分类 → 实体统计 → 工程量提取
5. 结果以卡片形式展示（图纸信息/图层列表/实体统计/审图建议）

### 2.3 对话功能

- 输入文字描述 → 按 Enter 或点击发送
- 支持自然语言：如"帮我生成一份投标文件"、"检查这个图纸的消防合规性"
- 对话历史自动保存

## 三、图纸审查

### 3.1 智能审图

上传图纸后，系统自动执行 15 条国标规则审查：

| 审查类别 | 规则数 | 检查内容 |
|---------|--------|---------|
| 消防疏散 | 3 | 疏散距离、安全出口数量、走道宽度 |
| 排烟竖井 | 2 | 排烟面积、间距 |
| 防火分区 | 2 | 防火分区面积、防火墙位置 |
| 结构安全 | 3 | 抗震设防、构件尺寸 |
| 电气安全 | 2 | 防雷接地、配电安全 |
| 给排水 | 2 | 消防给水、排水坡度 |
| 通用 | 1 | 标题栏标注完整性 |

审图结果以「通过/警告/错误」三级展示。

### 3.2 AI 改图 (DXF)

支持的编辑操作（12项）：
- 图层管理（新增/修改属性/删除）
- 文本编辑（修改内容/新增文本）
- 块属性编辑（修改TAG值）
- 实体操作（移动/复制/删除）
- 标注修改 / 颜色修改 / 线型修改

## 九、常见问题

**Q: 上传图纸后没反应？**
A: 检查文件格式是否为 DWG/DXF/PDF，单文件不超过套餐限制。

**Q: 审图结果不准确？**
A: 系统基于规则引擎自动审查，复杂图纸建议人工复核。

**Q: 如何找回密码？**
A: 登录页 → 忘记密码 → 通过绑定邮箱或安全问题重置。

**Q: 微信小程序打不开？**
A: 确认API服务器地址在白名单内，开发阶段需在微信开发者工具勾选「不校验合法域名」。

**Q: 如何注销账号？**
A: 联系管理员（boss_ke）处理。
