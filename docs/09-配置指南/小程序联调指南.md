# EMA 微信小程序联调指南

## 一、前置准备

### 1.1 安装工具

| 工具 | 下载地址 | 版本要求 |
|------|---------|---------|
| 微信开发者工具 | https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html | 最新稳定版 |
| HBuilderX | https://www.dcloud.io/hbuilderx.html | ≥3.0 |
| Node.js | https://nodejs.org | ≥16 |

### 1.2 获取微信 AppID

1. 打开 https://mp.weixin.qq.com
2. 注册小程序账号（个人/企业均可）
3. 开发 → 开发设置 → 获取 AppID
4. 将 AppID 填入 `mini-program/manifest.json` 的 `mp-weixin.appid` 字段

### 1.3 配置服务器域名

在微信小程序后台 → 开发 → 开发设置 → 服务器域名：

| 类型 | 域名 |
|------|------|
| request 合法域名 | `https://your-domain.com` |
| uploadFile 合法域名 | `https://your-domain.com` |

**本地调试**：微信开发者工具 → 详情 → 本地设置 → 勾选「不校验合法域名」

## 二、启动步骤

### 2.1 HBuilderX 方式（推荐）

1. 打开 HBuilderX
2. 文件 → 导入 → 从本地目录导入 → 选择 `mini-program/` 目录
3. 运行 → 运行到小程序模拟器 → 微信开发者工具

### 2.2 命令行方式

```bash
cd mini-program
npm install     # 首次运行

# 启动开发模式
npm run dev:mp-weixin
```

## 四、常见问题

**Q: uni.request 报错 "request:fail"**
A: 检查开发者工具是否勾选「不校验合法域名」，或确认 API_BASE 地址可访问。

**Q: wx.login 后无法获取用户信息**
A: 新版微信小程序需要通过 `<button open-type="getUserInfo">` 获取，不能用 `wx.getUserInfo` 直接调用。

**Q: 上传文件失败**
A: `uni.uploadFile` 的 `name` 参数需与后端一致（当前后端接收 `file` 字段）。

**Q: tabBar 图标不显示**
A: manifest.json 的 tabBar icon 路径需指向真实 PNG 文件，emoji 在 tabBar 不生效。
